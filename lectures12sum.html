FV – Future Value – ערך עתידי

הסכום הכולל הנצבר בעתיד, כולל הקרן וכל הריבית שנצברה. מבטא את תוצאת ההשקעה או ההלוואה בחלוף זמן.

🏦

PV – Present Value – ערך נוכחי

הסכום הראשוני בהווה (זמן 0) – סכום הפקדה או קרן הלוואה. נקרא גם "קרן מקורית".

⏳

ריבית פשוטה מול ריבית דריבית

ריבית פשוטה – מצוינת במפורש. הריבית מחושבת תמיד על הקרן המקורית בלבד.
ריבית דריבית (ברירת מחדל!) – אם לא נאמר "ריבית פשוטה", תמיד ריבית דריבית. הריבית נצברת גם על ריבית קודמת – שרשרת מכפלות → חזקה.

2ערך עתידי של סכום יחיד – נוסחאות
ריבית פשוטה · קבועה

הפקדה/הלוואה חד-פעמית, ריבית פשוטה אחת

מצויין במפורש "ריבית פשוטה". הריבית הכוללת = ריבית לתקופה × מספר תקופות.

FV = PV \cdot (1 + r \cdot n)r = ריבית תקופתית · n = מספר תקופות
ריבית פשוטה · משתנה

הפקדה/הלוואה חד-פעמית, ריביות פשוטות שונות

סוכמים את כל מכפלות ריבית × תקופות בסוגריים.

FV = PV \cdot (1 + r_1 n_1 + r_2 n_2 + \cdots)
ריבית דריבית · קבועה

הפקדה/הלוואה חד-פעמית, ריבית דריבית אחת

ברירת מחדל! מנגנון חזקה – ריבית על ריבית.

FV = PV \cdot (1 + r)^{n}ברירת מחדל כשלא מצוין "ריבית פשוטה"
=FV( rate, nper, 0, PV )
ריבית דריבית · משתנה

הפקדה/הלוואה חד-פעמית, ריביות דריבית שונות

מכפלת חזקות נפרדת לכל מקטע ריבית.

FV = PV \cdot (1+r_1)^{n_1} \cdot (1+r_2)^{n_2} \cdot \;\cdots

באקסל: עמודה נפרדת לכל ריבית – ראו חישוב רב-שלבי בהמשך.

📖 מקרא לנוסחאות

FV – Future Value – הערך העתידי (הסכום הכולל הנצבר בעתיד)
PV – Present Value – הערך הנוכחי (קרן ראשונית / סכום התחלתי)
r – Rate – שיעור הריבית לתקופה
n – מספר תקופות הריבית
3ערך עתידי של סדרה (PMT) – עקרונות
🔁

מהי סדרה? כלל סר"ת

סדרה מתקיימת כאשר שלושת התנאים מתקיימים יחד: סכום קבוע · ריבית קבועה · תדירות קבועה.
מילות מפתח: "בסוף כל חודש", "בתחילת כל שנה", "הפקדה חודשית קבועה".

כללים קריטיים בסדרות

rate ו-nper בסדרות

rate = ריבית לתקופת תשלום. משלמים כל חודש? ריבית חודשית. כל שנה? ריבית שנתית.

nper = מספר התשלומים בסדרה – לא מספר השנים!

pmt = סכום האיבר הקבוע בסדרה.
הפקדה / תשלום (כסף יוצא) ← סימן שלילי (−).
תקבול / משיכה (כסף נכנס) ← סימן חיובי (+).

[pv] = הפקדה חד-פעמית נוספת בהווה ← סימן שלילי (−), או הלוואה המתקבלת היום ← סימן חיובי (+). אם אין ← 0.

type – עיתוי הסדרה

מתי type = 1?

type = 0 (ברירת מחדל): תשלומים בסוף כל תקופה.

type = 1: תשלומים בתחילת כל תקופה, וקיים פער של תקופה אחת בין ההפקדה האחרונה למועד סיום הצבירה.

אם ההפקדה האחרונה חופפת למועד המשיכה ← type = 0, גם אם ההפקדות "בתחילת כל חודש"!

טיפ ספירת תשלומים: מספר התשלומים = עיתוי אחרון − עיתוי ראשון + 1.
לדוגמה: הפקדות בזמן 0 עד זמן 4 ← מספר תשלומים = 4 − 0 + 1 = 5.

5כללי זהב
⚠️

ברירת המחדל היא ריבית דריבית

לא כתוב "ריבית פשוטה" במפורש ← תמיד ריבית דריבית, אפשר להשתמש בפונקציות Excel. כתוב "ריבית פשוטה" ← נוסחה מתמטית בלבד.

✅

חפיפת יחידות: rate ↔ nper (תמיד!)

חייבת חפיפה בין יחידת הזמן של הריבית ל-nper: שנתית ← שנים, חודשית ← חודשים, חצי-שנתית ← חצאי שנים.
בסדרות: rate = ריבית לתקופת תשלום, nper = מספר התשלומים.

⚠️

סימני תזרימים (−/+)

הפקדות, תשלומים, כסף יוצא ← סימן שלילי (−).
הלוואות, תקבולים, משיכות, כסף נכנס ← סימן חיובי (+).

✅

אין מחלקים ריביות!

אסור לחלק ריבית שנתית ב-12 כדי לקבל חודשית (בגלל מנגנון ריבית דריבית). שיטות ייעודיות בהמשך.

🔗 חישוב רב-שלבי – כל שינוי = עמודה חדשה

כל שינוי (ריבית, סכום הפקדה, הפסקה, הפקדה חד-פעמית נוספת) = שלב חישוב נפרד (עמודה). כל שלב מסתיים במועד השינוי.

עיקרון המפתח: ה-FV שהתקבל בשלב מסוים הופך ל-PV של השלב הבא, בסימן הפוך:

7תהליך פתרון – תרשים החלטה

🔍 שלב 1 – זיהוי הנדרש

מה ידוע? מה הנעלם? צריך לחשב FV, או ש-FV נתון ויש לחלץ rate / nper / pmt / pv?

📋 שלב 2 – סיווג הריבית

כתוב "ריבית פשוטה"? ← נוסחה מתמטית בלבד. לא כתוב? ← ריבית דריבית ← Excel.

📌 שלב 3 – סדרה או סכום יחיד?

מתקיים סר"ת? ← PMT ≠ 0 (ריבית לתקופת תשלום, nper = מספר תשלומים). לא ← PMT = 0.

🔢 שלב 4 – כמה שלבי חישוב?

אין שינויים ← שלב אחד. יש שינוי (ריבית / סכום / הפקדה נוספת / הפסקה) ← עמודה לכל מקטע, כשכל שלב מסתיים במועד השינוי.

✅ שלב 5 – ביצוע

הזנת הפרמטרים בכל שלב. FV של כל שלב ← PV בסימן הפוך לשלב הבא.

8בחירה בין חלופות על סמך ערך עתידי
⚖️

השוואה חייבת להיות באותה נקודת זמן

מחשבים FV לכל חלופה בנפרד לאותו מועד. חלופות הכנסה ← עדיפה FV גבוה. חלופות עלות ← עדיפה FV נמוך.

🔀

ריבית אדישות

הריבית שבה שתי חלופות שוות ערך. מציבים PV = חלופה אחת, FV = חלופה שניה, ומחלצים rate.

שימו לב: כמוסבר במפגשים 3–4, ניתן לבחור בין חלופות גם על בסיס ערך נוכחי (PV).
